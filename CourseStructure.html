Python Data Engineering - 12-Day Training Program

Course Overview

This 12-day intensive training covers essential Python skills for Data Engineering. Each session lasts 4 hours and includes hands-on practice, real-world case studies, and challenging assignments to solidify learning.

Curriculum

Day Topics Key Concepts Assignments & Projects
Day 1 Python Basics Variables, Data Types, Loops, Functions, File Handling, Exception Handling Process a CSV file, Read & manipulate JSON data, Implement a CLI calculator
Day 2 SQL Fundamentals Database Design, SELECT, INSERT, UPDATE, DELETE, Joins, Aggregations Create a database, Write optimized queries, Design a relational schema
Day 3 Advanced SQL Stored Procedures, Indexing, Query Optimization, Transactions, ACID Properties Develop a stored procedure, Optimize slow queries, Implement transactions
Day 4 NumPy Arrays, Indexing, Broadcasting, Vectorized Operations, Linear Algebra Matrix transformations, Statistical analysis, Image processing with NumPy
Day 5 Pandas Basics DataFrames, Series, Reading/Writing Files, Filtering, Handling Missing Data Process large datasets, Clean & filter data, Perform exploratory data analysis
Day 6 Advanced Pandas Merging, GroupBy, Pivot Tables, Time Series Analysis, MultiIndexing Aggregate datasets, Build pivot tables, Analyze stock market data
Day 7 Excel Automation OpenPyXL, Pandas Excel Integration, Formatting, Generating Reports Automate Excel reports, Create dynamic charts, Convert CSV to formatted Excel
Day 8 Database Integration SQLAlchemy, PostgreSQL, Query Execution, Connection Pools Develop an ETL pipeline, CRUD operations, Automate data ingestion
Day 9 AWS S3 Boto3, Upload/Download Files, Presigned URLs, Bucket Policies Upload & retrieve data, Implement security policies, Automate backups
Day 10 SharePoint & Network Drive Office365 API, PyDrive, Authentication, Permissions Automate file management in SharePoint, Sync local files with cloud storage
Day 11 Working with APIs REST APIs, Authentication, Requests Library, Handling JSON/XML Data Fetch real-time data, Build a mini API, Implement OAuth authentication
Day 12 Project & Best Practices ETL Pipeline, Error Handling, Performance Tuning, Deployment Develop an ETL pipeline, Debug an existing project, Deploy on AWS Lambda

Challenging Assignments

  • SQL Optimization: Write complex queries to process large datasets efficiently.
  • ETL Pipeline: Build an end-to-end pipeline from API to database storage.
  • API Integration: Develop a Python API to fetch and process real-time data.
  • AWS S3 Data Handling: Implement an automated backup system using S3.
  • Data Visualization: Generate interactive reports from real-world datasets.
  • Database Migration: Migrate data from a relational DB to a NoSQL database.
  • Excel Report Automation: Automate a dynamic Excel dashboard using Python.
  • Stock Market Analysis: Use Pandas and NumPy to analyze and visualize stock trends.
  • Fraud Detection System: Implement an anomaly detection system using Python.
  • Social Media Data Mining: Scrape and analyze Twitter data using APIs.
  • Data Pipeline Performance Benchmarking: Optimize data processing speed.
  • Real-time Data Streaming: Set up a pipeline for streaming data from APIs to databases.
